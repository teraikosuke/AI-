# LINE公式アカウント 料金プラン・メッセージ仕様 v1

## 1. 本書の目的
本資料は、「CELM様 AIペルソナプロジェクト」におけるLINEボット開発で利用するLINE公式アカウントの料金プラン、およびメッセージ通数のカウント仕様について解説するものである。
特に、タイムアウト対策で採用するアーキテクチャが運用コストに与える影響を明確にすることを目的とする。

## 2. メッセージAPIの種類と課金仕様
LINEボットがユーザーに応答を返すAPIには主に2種類あり、それぞれ課金仕様が大きく異なる。

| APIの種類 | 応答タイミング | 課金対象 | 特徴 |
| :--- | :--- | :--- | :--- |
| **Reply Message** | 同期応答 | **無料** | ユーザーからのメッセージに対し1度だけ応答可能。応答トークンの有効期限が短いため、時間のかかる処理には不向き。 |
| **Push Message** | 非同期応答 | **有料** | ボット側から好きなタイミングで送信可能。月間メッセージ通数としてカウントされ、プランの無料枠を超えると課金対象となる。 |

**本プロジェクトでは、LLMの応答生成に時間がかかるため、タイムアウトを回避するべく `Push Message` を採用する。したがって、ユーザーからの質問に対するすべての回答が、月間のメッセージ通数としてカウントされる。**

## 3. 料金プラン比較
LINE公式アカウントには、メッセージ配信数に応じた3つの料金プランがある。（2025年10月時点）

| プラン名 | 月額固定費（税抜） | 無料メッセージ通数 | 追加メッセージ料金（税抜） |
| :--- | :--- | :--- | :--- |
| **コミュニケーションプラン** | 0円 | 200通 | 送信不可 |
| **ライトプラン** | 5,000円 | 5,000通 | 送信不可 |
| **スタンダードプラン** | 15,000円 | 30,000通 | 3円〜/通（従量課金） |

*出典: [LINE for Business 公式サイト](https://www.linebiz.com/jp/service/line-official-account/plan/)*

## 4. プロジェクトにおけるプラン選定の指針

### 4.1. トライアル期間中の利用量試算
- **対象ユーザー**: 25名
- **想定メッセージ数**: ユーザー1人あたり月間平均 **X** 回の質問をすると仮定
- **月間合計メッセージ数**: `25名 × X回 = 25X 通`

| 1人あたりの月間質問数 (X) | 月間合計メッセージ数 | 推奨プラン | 月額費用（税抜） |
| :--- | :--- | :--- | :--- |
| **〜8回** | 〜200通 | コミュニケーションプラン | 0円 |
| **9回〜200回** | 201〜5,000通 | ライトプラン | 5,000円 |
| **201回〜** | 5,001通〜 | スタンダードプラン | 15,000円〜 |

### 4.2. 推奨事項
- **トライアル開始初期**: まずは**コミュニケーションプラン（無料）**で開始し、実際の利用状況（メッセージ通数）を監視する。
- **プランの切り替え**: 月間200通を超過しそうになった時点で、速やかに**ライトプラン**へアップグレードする。トライアルの利用頻度によってはスタンダードプランが必要になる可能性も視野に入れる。
- **モニタリング**: LINE Official Account Managerの管理画面で、日々のメッセージ配信数を注視する必要がある。

## 5. 注意事項
- 本資料に記載の料金は、2025年10月1日時点のものです。最新の情報はLINEの公式サイトをご確認ください。
- プランのダウングレードは翌月適用となるため、アップグレードは慎重に検討する必要があります。
