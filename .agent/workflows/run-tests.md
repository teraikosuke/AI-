# テスト実行 (Run Tests)

## 実行 // turbo
テストを実行せよ：
```bash
pnpm test || npm test || yarn test || pytest
```

## 結果判定
- **成功**: 全テスト通過 → `PASS` を返す
- **失敗**: 失敗したテストあり → `FAIL` を返す
- **スキップ**: スキップされたテストあり → `PASS_WITH_SKIPPED` を返す

## 出力内容
以下を報告せよ：
- 実行したテスト数
- 成功/失敗/スキップの内訳
- 失敗したテスト名と理由（最大5件）
- 実行時間
